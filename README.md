# 🕉️ MitSara Nexus v20.0
### The Ultimate Intelligence Swiss Army Knife for LLM Datasets

MitSara Nexus is a high-performance, mobile-first autonomous pipeline designed to transform raw knowledge (PDF, TXT, MD) into structured, training-ready datasets for Llama-3, Mistral, and RAG architectures.

## 🚀 Key Features
- **Autonomous Semantic Chunking:** Automatically splits long-form documents into logical segments.
- **Multi-Framework Support:** Native templates for **CoT, ToT, ReAct, RACE, and RAG**.
- **Template Injection:** One-tap formatting for Llama-3, Mistral, and Gemma chat templates.
- **Swiss Army Knife Tools:**
    - **CSV Transformer:** Convert raw CSV rows to JSON datasets instantly.
    - **Dataset Statistician:** Real-time token estimation and entry counts.
    - **Custom Logic Loader:** Teach the engine new structures using seed JSON.
- **Tiranga UI:** A beautiful, responsive Indian Tricolor-inspired interface with high-end CSS animations.

## 🛠️ Deployment
1.  **Clone** this repository.
2.  **Upload** `index.html` and `vercel.json` to your GitHub.
3.  **Connect** to Vercel and hit **Deploy**.

## 🛡️ Privacy
All processing happens **locally** in your browser via `pdf.js`. Your data never leaves your device unless you choose to export it.
